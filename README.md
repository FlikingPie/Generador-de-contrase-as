# 🔐 Generador de Contraseñas - CustomTkinter

Aplicación de escritorio desarrollada en **Python** utilizando **CustomTkinter** para generar contraseñas seguras de manera rápida y configurable.

El programa permite seleccionar diferentes tipos de caracteres y generar contraseñas aleatorias que se almacenan automáticamente en un archivo.

---

# 📌 Características

- Interfaz gráfica moderna con **CustomTkinter**
- Generación de contraseñas seguras usando **secrets**
- Configuración de longitud de contraseña
- Inclusión opcional de:
  - 🔢 Números
  - 🔣 Símbolos
- Uso automático de:
  - Letras mayúsculas
  - Letras minúsculas
- Barra de carga durante la generación
- Guardado automático en archivo **Contraseñas.txt**
- Validación de entrada de usuario
- Notificación con hora de generación

---

# 🖥️ Interfaz

La aplicación incluye:

- Campo para ingresar número de caracteres
- Checkboxes para seleccionar tipos de caracteres
- Botón para generar contraseña
- Ventana de carga
- Notificaciones emergentes
